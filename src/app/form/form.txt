types of forms
1. Template driven Forms{
    simple basic form
    easy to StructuralComponent
    based on template(HTML)
}
2.Reactive Forms{
    complex form with greater control
    we defines structure of form in ts 
    based on typescript
}

NgForm create object of form while using template reference variable

ngModel tell the define control or input field is the the part of form

setValue and patchValue are two way to set value of form or input field

this.myform.reset(); is used for reset input field and their state
